autograd is a pure Go automatic differentiation library inspired by [deep-learning-from-scratch-3](https://github.com/oreilly-japan/deep-learning-from-scratch-3). It lets you define mathematical functions over tensors and compute their gradients automatically — enabling gradient descent, neural network training, and higher-order differentiation entirely in Go.

## Get started

<Steps>
  <Step title="Install the package">
    Add autograd to your Go module:
    ```bash
    go get github.com/itsubaki/autograd
    ```
  </Step>
  <Step title="Create a variable and compute a gradient">
    ```go
    import (
        F "github.com/itsubaki/autograd/function"
        "github.com/itsubaki/autograd/variable"
    )

    x := variable.New(1.0)
    y := F.Sin(x)
    y.Backward()

    fmt.Println(y)      // variable(0.8414709848078965)
    fmt.Println(x.Grad) // variable(0.5403023058681398)
    ```
  </Step>
  <Step title="Train a model">
    Use built-in layers, models, and optimizers to train neural networks on your data. See the [Deep Learning guide](/guides/deep-learning) for a complete walkthrough.
  </Step>
</Steps>
